# FinalMate 产品需求说明

## 一、产品定位

FinalMate 是面向大学生期末考试场景的 AI 学习 Agent。它的目标不是替代所有学习过程，而是在考试前有限时间里，通过资料解析、考点判断、主动推进、讲解、记忆、练习和反馈，帮助学生更高效地完成应试复习。

## 二、核心用户痛点

1. 复习资料分散：PPT、教材、笔记、作业、老师划重点、往年卷分布在不同地方。
2. 重点判断困难：学生不知道哪些章节更可能考、哪些题型更重要。
3. 网课效率低：网课线性播放，无法根据学生薄弱点动态调整。
4. 用户不会提问：很多学生不知道该怎么问 AI，学习容易中断。
5. 背诵效率低：概念、公式、流程、简答题需要不同记忆策略。
6. 缺少检测闭环：看懂不代表会做题，学生需要持续出题与批改。

## 三、核心功能

### 1. 资料解析 Agent

输入课程 PPT、教材、笔记、作业、老师划重点、往年卷。输出章节结构、知识点、题型、易错点、教师强调内容。

### 2. 考点分析 Agent

如果存在往年卷，则统计知识点出现频率、题型分布、分值占比，生成考点热力图和优先级。

### 3. 人类化讲解 Agent

讲解结构：

- 这是什么
- 为什么重要
- 考试怎么考
- 用一个例子讲懂
- 典型题怎么做
- 常见错误是什么

### 4. 阶段性主动推进 Agent

系统不只是等待用户提问，而是在关键节点主动询问：

- “继续学习 XXX 吗？”
- “是否用一道例题巩固？”
- “要不要进入 3 道小测？”
- “是否把这个知识点加入思维导图？”
- “你刚才答错了，要不要换一种讲法？”

### 5. 知识可视化 Agent

展示：

- 复习进度
- 掌握度
- 考点热力图
- 错题分布
- 思维导图
- 知识链条

### 6. 记忆方法 Agent

根据知识类型给策略：

- 概念类：口诀、类比、对比表
- 公式类：符号拆解、推导逻辑
- 流程类：步骤图、顺口溜
- 混淆类：对照卡片
- 简答题：关键词骨架 + 分点展开

### 7. 出题与批改 Agent

生成选择题、填空题、简答题、计算题、案例题和模拟卷，并给出评分、失分点、参考答案和下一步建议。

## 四、MVP 范围

首版支持：

- 一个课程 Demo
- 手动/模拟上传资料
- 往年卷统计 Mock
- 复习路径排序
- 主动推进按钮
- 思维导图创建
- 自动出题与批改 Mock
- 本地前后端联调
